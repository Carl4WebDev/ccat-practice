# Cognitive Aptitude Practice Tool

A simple web-based application designed to simulate timed cognitive aptitude tests similar to CCAT-style assessments.

## Purpose

This project was built to improve:

- Speed in solving problems
- Accuracy under time pressure
- Logical and pattern recognition skills

## Features

- Timed test environment (15 minutes)
- Multiple difficulty levels (Easy, Hard, Extreme)
- Question navigation and flagging
- Score breakdown and review

## In Progress

- Visual abstract reasoning (SVG-based pattern questions) is currently under development and not yet fully implemented.

## Tech Stack

- React (Vite)
- Tailwind CSS
- SVG (planned for visual pattern rendering)

## Usage

1. Select a difficulty level
2. Answer questions within the time limit
3. Review your results and performance

## Disclaimer

This is an independent practice tool and is not affiliated with or endorsed by any official testing provider.
